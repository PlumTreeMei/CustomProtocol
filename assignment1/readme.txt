File Structure:
The custom protocol is defined in protocol.h and protocol.c.
server.c and client.c stores the source code of the programs that run on server and client.
noresponse.c simluated a server that never responds.

Steps to run:
1)use makefile to complie the files.You can use the command make in terminal.
2)use command ./server to run the server(use ./no response to run a server that never responds) in terminal
3)open another terminal and use command ./client to run the client
